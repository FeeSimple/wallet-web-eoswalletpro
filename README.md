# feesimple.io/wallet

A secure, native eos wallet for storing eos and tokens that follow the eosio.token standard.

# How to build locally on unix

```
git clone https://github.com/Hashbit-Technologies/eoswalletpro
cd eoswalletpro
npm install
cd eos-pro/eosjs
npm install
cd ..
cd ..
node app
```

Now you can visit: 
http://localhost:3000
http://127.0.0.1:3000
